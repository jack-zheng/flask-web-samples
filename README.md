# Flask-Web-Samples
This repo is basically for practicing python code when reading flask related guide book.

The books I refered:

* Flask Web 实战 - 李辉

## Env Setup Tips

**[Linux like]**

```
$ export FLASK_APP=hello
$ flask run
```

**[Windows OS]**

CMD mode:

```
> set FLASK_APP=run
> flask run
```

Power Shell mode:

```
> $env:FLASK_APP="run"
> flask run
```
